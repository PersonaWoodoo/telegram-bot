## 👑 Админ-команды

### Выдача и списание
| Команда | Что делает |
|---------|-----------|
| `/give ID gram 1000` | Выдать пользователю Gram |
| `/give ID gold 50` | Выдать пользователю Gold |
| `/take ID gram 500` | Забрать у пользователя Gram |
| `/take ID gold 25` | Забрать у пользователя Gold |

### Информация и управление
| Команда | Что делает |
|---------|-----------|
| `/info @username` | Информация о пользователе по юзернейму |
| `/info ID` | Информация о пользователе по ID |
| `/setadmin ID` | Выдать права администратора |
| `/removeadmin ID` | Снять права администратора |
| `/ban ID` | Заблокировать пользователя |
| `/unban ID` | Разблокировать пользователя |

### Статистика и логи
| Команда | Что делает |
|---------|-----------|
| `/stats` | Статистика бота (пользователи, ставки, пополнения) |
| `/logs` | Логи действий администраторов |
| `/all` | Список всех пользователей бота |

### Рассылка и промокоды
| Команда | Что делает |
|---------|-----------|
| `/broadcast` | Сделать рассылку всем пользователям |
| `/addpromo КОД ГРАММЫ ГОЛД АКТИВАЦИИ` | Создать промокод |

### Обработка заявок
| Команда | Что делает |
|---------|-----------|
| `/approve_transfer ID` | Подтвердить заявку на пополнение |
| `/decline_transfer ID` | Отклонить заявку на пополнение |
| `/approve_withdraw ID` | Подтвердить заявку на вывод |
| `/decline_withdraw ID` | Отклонить заявку на вывод |

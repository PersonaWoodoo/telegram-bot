# WILLD GRAMM Casino Bot

Telegram бот с играми, пополнением через Stars и системой валют.

## Команды

### Пользовательские
- `/start` - запуск бота
- `/cancel` - отмена активной игры
- `б` / `баланс` - показать баланс
- `профиль` - показать профиль
- `топ` - топ игроков
- `бонус` - ежедневный бонус
- `реф` - реферальная система
- `чеки` - работа с чеками
- `промокод КОД` - активировать промокод

### Игры (текстовые команды)
- `золото 100 gram gold` - игра Золото
- `алмазы 100 gram diamond` - игра Алмазы
- `кубик 100 gram 5` - игра Кубик
- `кости 100 gram больше` - игра Кости
- `футбол 100 gram гол` - игра Футбол
- `баскет 100 gold точный` - игра Баскетбол

### Админ-команды
- `/give ID gram 1000` - выдать валюту
- `/take ID gram 500` - забрать валюту
- `/info @username` - информация о пользователе
- `/setadmin ID` - выдать админа
- `/removeadmin ID` - снять админа
- `/ban ID` - заблокировать
- `/unban ID` - разблокировать
- `/stats` - статистика бота
- `/logs` - логи админов
- `/broadcast` - рассылка
- `/addpromo КОД ГРАММЫ ГОЛД АКТИВАЦИИ` - создать промокод
- `/approve_deposit ID` - подтвердить пополнение
- `/decline_deposit ID` - отклонить пополнение
- `/approve_withdraw ID` - подтвердить вывод
- `/decline_withdraw ID` - отклонить вывод

## Курс Stars
1 Star = 0.70 Gold (можно менять в админ-панели)

## Защита от флуда
3 секунды между нажатиями кнопок

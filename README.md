# Команды бота

## Пользовательские команды
/start — Запуск бота
/cancel — Отмена активной игры

## Админ-команды
/give ID gram 1000 — Выдать Gram
/give ID gold 50 — Выдать Gold
/take ID gram 500 — Забрать Gram
/take ID gold 25 — Забрать Gold
/info @username или ID — Информация о пользователе
/setadmin ID — Выдать права администратора
/removeadmin ID — Снять права администратора
/ban ID — Заблокировать пользователя
/unban ID — Разблокировать пользователя
/stats — Статистика бота
/logs — Логи действий админов
/all — Список всех пользователей
/broadcast — Рассылка
/addpromo КОД ГРАММЫ ГОЛД АКТИВАЦИИ — Создать промокод
/approve_transfer ID — Подтвердить пополнение
/decline_transfer ID — Отклонить пополнение
/approve_withdraw ID — Подтвердить вывод
/decline_withdraw ID — Отклонить вывод
